# Push to GitHub

The citation checker is committed locally. Push it to your repo with:

```bash
cd /Users/admin0513-27/CVE-2025-5777/citation_checker
git push -u origin main
```

If you use **HTTPS** and Git asks for credentials:
- Use your GitHub username and a **Personal Access Token** (not your password).
- Create a token: GitHub → Settings → Developer settings → Personal access tokens.

If you use **SSH** instead:
```bash
git remote set-url origin git@github.com:rashedhasan090/citationcheckertool.git
git push -u origin main
```

Repo URL: **https://github.com/rashedhasan090/citationcheckertool**
